The "Prop Drilling" Problem

Passing data through 10 layers of components just to reach the bottom one is messy. This is called "Prop Drilling".

The Solution: Context API

Context allows you to "teleport" data. You wrap a section of your app in a Provider, and any component inside can Consume that data directly. 

// 1. Create
const UserContext = createContext();

// 2. Provide
<UserContext.Provider value={user}>
   <Dashboard />
</UserContext.Provider>

// 3. Consume (Deep inside Dashboard)
const user = useContext(UserContext);

Reusable Logic: Custom Hooks

If you find yourself copying and pasting useEffect code between components, you should extract it into a Custom Hook.

Custom Hooks are just JS functions that start with use. They can call other hooks.

  • useFetch
  • useWindowSize
  • useAuth
// useWindowWidth.js
function useWindowWidth() {
  const [width, setWidth] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth);
    window.addEventListener('resize', handleResize);
    return () => window.removeListener('resize', handleResize);
  }, []);

  return width;
}

React + TypeScript

JavaScript is loosely typed, which leads to "undefined is not a function" crashes. TypeScript enforces rules on Props and State.

Why use it?

It acts as live documentation. You know exactly what props a component requires just by hovering over it. 

interface ButtonProps {
  label: string;
  onClick: () => void;
  // Optional prop
  variant?: 'primary' | 'secondary'; 
}

const Button = ({ label, onClick }: ButtonProps) => {
  return <button onClick={onClick}>{label}</button>
}

Performance: Memoization

React re-renders components frequently. Sometimes, calculations are expensive.

We use Memoization to cache results so we don't recalculate them unless necessary.

useMemo

Caches a value (e.g., filtering a list of 10,000 items).

const sorted = useMemo(() => list.sort(), [list])

useCallback

Caches a function definition. Useful when passing functions to children to prevent them from re-rendering.
